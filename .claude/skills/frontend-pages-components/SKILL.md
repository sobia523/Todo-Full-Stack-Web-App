# Frontend Pages & Components Skill

## Instructions

1. **Page Structure**
   - Semantic HTML5 structure
   - Header, main, footer separation
   - Clear content hierarchy

2. **Components**
   - Reusable UI components (buttons, cards, forms)
   - Props-based or class-based variants
   - Consistent spacing and sizing

3. **Layout**
   - Flexbox and Grid usage
   - Responsive breakpoints
   - Alignment and spacing system

4. **Styling**
   - Modern CSS (Flexbox, Grid, variables)
   - Utility-first or component-based styles
   - Hover, focus, and active states

## Best Practices
- Mobile-first design
- Reusable and modular components
- Consistent color and typography system
- Avoid inline styles
- Keep components small and focused
